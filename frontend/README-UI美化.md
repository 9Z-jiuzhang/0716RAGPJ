# 前端 UI 美化说明

> 范围：在蓝白配色与 5.1 功能不变的前提下，优化壳层布局与观感。  
> 对应：产品手册 **4.3**（配色 / 组件 / 确认弹窗 / 管理端快捷入口）、**5.1**（访客端 / 管理端）。  
> 主色固定：`#1A73E8` / `#4A90D9` / `#0D47A1`。  
> **布局（本迭代）**：**顶栏导航 + 全宽内容**（已取消左右侧栏）。

---

## 1. 原则与红线

| 必须遵守 | 明确不做 |
|----------|----------|
| 蓝白配色，主色三元组不变 | 默认暗色主题 |
| 顶栏主导航 + 内容区全宽/居中 | 恢复深蓝左右侧栏 |
| 简洁现代、圆角卡片、浅色背景 | 紫/靛渐变、奶油底衬线、霓虹发光 |
| 危险操作确认弹窗 | 访客导航塞管理项 |
| 管理端右上「智能对话」「退出」 | 彩色 pill 堆叠、多层彩色阴影 |

### 与手册 §4.3「左导航」的说明

手册原文为「左侧导航栏 + 右侧内容区」。本迭代按专业控制台习惯调整为 **顶栏布局**，配色与确认弹窗、管理端「智能对话 / 退出」等仍对齐 4.3；合并前建议与组长确认布局偏离是否接受。

## 2. 布局方案（当前实现）

```text
┌─────────────────────────────────────────────────────────┐
│ 品牌 │ 导航链接（下划线选中）…… │ 用户信息 / 快捷按钮 │  ← topnav
├─────────────────────────────────────────────────────────┤
│ 当前页标题                                                 ← page-bar
├─────────────────────────────────────────────────────────┤
│                                                         │
│              内容区（max-width 1280；问答页放宽居中）        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

| 端 | 顶栏结构 |
|----|----------|
| 访客（未登录） | 品牌｜智能问答｜登录 |
| 注册用户 | 品牌｜智能问答 / 历史 / 个人中心｜退出 |
| A/B 部门员工 | 再加「文档上传」（目标库按部门隔离） |
| 普通/超级管理员 | `/admin/` 顶栏按权限码裁剪菜单；超管含用户/角色/模型写入 |
| 登录页 | **无业务顶栏**，独立全屏 + 登录/注册 Tab |

- 选中态：主色文字 + 底边 2px `#1A73E8`（白顶栏，不用整条深蓝底）。  
- 平板：顶栏换行，导航第二行横向滚动。  
- 问答页：`content-wide` + 消息列约 880px 居中，对话优先。

## 4. 页面观感要点

### 4.1 访客端与角色分流

- **登录页独立全屏**（`#/login`），注册为页内 Tab。  
- **访客**：仅问答公开库。  
- **注册用户 `user`**：问答 + 历史 + 个人中心。  
- **A/B 部门员工**（`staff_a` / `staff_b`）：对齐手册 kb_admin + §3.4 知识库隔离，仅本部门库可上传/维护。  
- **普通管理员 `admin`**：进管理端；无用户/角色写入、无模型写入。  
- **超级管理员 `super`**：全量权限（含用户角色与模型）。  
- 演示横幅：浅蓝信息条。

### 4.2 管理端

- 仪表盘：统计卡一行 → 趋势双列 → 资源 / 错误 / 队列。  
- 列表：工具条卡片 + 表格；危险操作确认弹窗。  
- Grafana：占位说明，不伪造图表。

### 4.3 组件与动效

| 项 | 约定 |
|----|------|
| 圆角 | 卡片/输入 8px；气泡 12px；弹窗 10px |
| 阴影 | 边框为主、轻阴影为辅 |
| 表格 | 表头浅灰；行悬停淡蓝；状态徽章 |
| 动效 | 顶栏选中 / 消息出现 / 按钮 hover ≈ 150ms |

## 5. 落地文件清单

| 文件 | 改动内容 |
|------|----------|
| `shared/css/layout.css` | **顶栏壳层**（`.topnav` / `.page-bar` / `.content`），移除侧栏布局 |
| `shared/css/variables.css` | 设计令牌；`--sidebar-width` 仅兼容保留 |
| `shared/css/components.css` | 按钮、表格、弹窗、演示横幅等 |
| `shared/css/base.css` | 全局排版 |
| `guest/css/guest.css` | 问答居中列、引用区、输入条 |
| `admin/css/admin.css` | 统计卡、趋势图、工具条、Grafana 占位 |
| `guest/js/app.js` | 独立登录+注册 Tab；按角色导航与跳转 |
| `admin/js/app.js` | 管理端门禁（需管理员权限） |
| `shared/js/auth.js` | `getPrimaryRole` / `canUpload` / `getPostLoginTarget` |
| `shared/js/mock.js` | 演示账号 admin / staff / user |

**不改**：路由、权限逻辑、API 契约、手册要求的功能入口。
