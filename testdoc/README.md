# 系统上传测试文档说明

本目录用于验证文档上传 → 解析 → 分段 → 向量化 → 问答检索全链路。

## 支持的文档类型（与系统一致）

系统当前支持：**PDF / DOC / DOCX / TXT / Markdown**。  
（CSV / XLSX / PPTX 会被拒绝，本目录不提供。）

| 文件 | 类型 | 说明 |
|------|------|------|
| `01_年假与加班制度.txt` | TXT | UTF-8 纯文本；年假/调休/加班规则 |
| `02_差旅与报销指引.md` | MD | 含标题与表格；差旅审批与报销标准 |
| `03_信息安全与账号权限规范.docx` | DOCX | Word 新格式；账号权限与密码策略 |
| `04_信息安全策略_PDF测试.pdf` | PDF | 可抽取英文正文（验证 PDF 解析） |
| `05_DOC解析样例_入职须知.doc` | DOC | 真实 OLE `.doc` 样例（旧版 Word） |
| `06_入职试用期手册_docx结构.doc` | DOC* | 扩展名 `.doc`、内容为 docx 结构（兼容分支） |

\* `06` 用于覆盖「误标扩展名」解析路径；`05` 用于真实旧版 DOC。

## 建议测试步骤

1. 使用 `admin` 或 `staff_a` 登录管理端 / 访客端上传页。
2. 选择一个可写知识库，依次上传上表全部文件。
3. 等待状态变为 `ready`。
4. 在问答页提问，例如：
   - 「入职满一年有几天年假？」
   - 「一线城市差旅住宿标准是多少？」
   - 「生产库只读权限要谁审批？」
   - 「试用期一般多久？转正前几天评估？」

## 重新生成 Office 文件

若需重建 DOCX/PDF：

```bash
docker compose run --rm -v "${PWD}/testdoc:/out" api python /out/_generate_office.py
```
